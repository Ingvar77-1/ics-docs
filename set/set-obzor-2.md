# Сеть

В Шлюзе безопасности ИКС есть множество сетевых инструментов, которые можно использовать для создания, контроля и обслуживания сетей.

---

**Мастер настройки сети**

Позволяет быстро и просто [настроить](https://doc.a-real.ru/index.php?article=54) основные сети в ИКС. Его рекомендуется использовать при первоначальной настройке.

**Провайдеры и сети**

В данном модуле можно [управлять](https://doc.a-real.ru/index.php?article=55) всеми сетями, которые контролируются ИКС: локальными сетями, провайдерами, VPN-сетями и туннелями, DMZ-сетями и т. п.

**Сетевое окружение**

Позволяет [настроить](https://doc.a-real.ru/index.php?article=56) участие ИКС в рабочих группах и доменах Windows.

**Сетевые утилиты**

Предоставляет [средства анализа сетей](https://doc.a-real.ru/index.php?article=57) при помощи таких популярных утилит, как ping, tracert, тест скорости канала и прочие.

**Межсетевой экран**

Это основное средство защиты для любого сетевого устройства. Контролирует трафик по портам и IP-адресам. В данном модуле можно [настроить](https://doc.a-real.ru/index.php?article=27) доступ, правила и NAT.

**Маршруты**

Движение трафика в сетях идет по определённым [маршрутам](https://doc.a-real.ru/index.php?article=58). Если ваша сеть сложнее стандарта (локальная сеть > ИКС > Интернет), необходимо прописать на ИКС, куда для какой сети отправлять пакеты.

**DNS**

Отвечает за сопоставление доменных имён с IP-адресами. Можно [настроить серверы](https://doc.a-real.ru/index.php?article=60), у которых ИКС будет запрашивать DNS-данные, а также создать свои DNS-зоны.

**DHCP**

Позволяет раздавать IP-адреса и сопутствующие данные устройствам в локальной сети автоматически, исходя из того, что указано в ИКС. Если [DHCP](https://doc.a-real.ru/index.php?article=61) не используется, настройка каждого устройства должна производиться вручную.

**Прокси**

Данный модуль в ИКС отвечает за [использование и контроль](https://doc.a-real.ru/index.php?article=62) HTTP/HTTPS-трафика.

**VPN**

Позволяет [управлять](https://doc.a-real.ru/index.php?article=63) защищённым подключением пользователей извне.

**ARP-таблица**

Показывает [соответствие](https://doc.a-real.ru/index.php?article=64) логических IP-адресов физическим MAC-адресам. Используется для тонкого анализа работы сети, имеет возможность жёсткой привязки IP-адреса к MAC.

**Удалённое управление**

Позволяет [настроить управление](https://doc.a-real.ru/index.php?article=65) несколькими ИКС из одного интерфейса.
