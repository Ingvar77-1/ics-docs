Модуль «Маршруты» предназначен для просмотра и редактирования маршрутов межсетевого экрана ИКС.

---

Модуль **«Маршруты»** предназначен для просмотра и редактирования [маршрутов](dobavit-marshrut-2.md) [межсетевого экрана](../../o-dokumentacii/slovar-terminov-3.md) и расположен в меню **Сеть &gt; Маршруты**.

![Иллюстрация](../../images/article_205/routes1-2.png)

На странице модуля отображаются списки статических и условных маршрутов, созданных в ИКС. Сюда входят маршруты, созданные в [правилах межсетевого экрана](../mezhsetevoy-ekran/mezhsetevoy-ekran-obzor-3.md), и маршруты, добавленные через модуль «Маршруты».

В модуле можно управлять маршрутами ([добавлять](dobavit-marshrut-2.md), редактировать, удалять, выключать) при помощи соответствующих функциональных кнопок.

Маршруты необходимы, чтобы задать правила, какой трафик по какому направлению движется. Например, у вас есть медленный, но стабильный провайдер и быстрый, но с обрывами. С помощью маршрутов можно задать, чтобы телефония, для которой нужно стабильное соединение, шла по первому каналу, а [HTTPS](../../o-dokumentacii/slovar-terminov-3.md)-трафик — по второму.
