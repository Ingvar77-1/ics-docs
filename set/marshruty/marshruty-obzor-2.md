# Маршруты

Модуль «Маршруты» предназначен для просмотра и редактирования маршрутов межсетевого экрана ИКС.

---

Модуль **«Маршруты»** предназначен для просмотра и редактирования [маршрутов](https://doc.a-real.ru/index.php?article=58) [межсетевого экрана](https://doc.a-real.ru/index.php?article=24#firewall) и расположен в меню **Сеть &gt; Маршруты**.

![Иллюстрация](../../../images/article_205/routes1-2.png)

На странице модуля отображаются списки статических и условных маршрутов, созданных в ИКС. Сюда входят маршруты, созданные в [правилах межсетевого экрана](https://doc.a-real.ru/index.php?article=27#tab3), и маршруты, добавленные через модуль «Маршруты».

В модуле можно управлять маршрутами ([добавлять](https://doc.a-real.ru/index.php?article=58), редактировать, удалять, выключать) при помощи соответствующих функциональных кнопок.

Маршруты необходимы, чтобы задать правила, какой трафик по какому направлению движется. Например, у вас есть медленный, но стабильный провайдер и быстрый, но с обрывами. С помощью маршрутов можно задать, чтобы телефония, для которой нужно стабильное соединение, шла по первому каналу, а [HTTPS](https://doc.a-real.ru/index.php?article=24#https)-трафик — по второму.
