На главной странице веб-интерфейса управления ИКС расположены меню модулей, настраиваемые виджеты и панель пользователя.

---

На главной странице веб-интерфейса управления ИКС расположены:

- слева — [меню](menyu.md) в виде группированного списка [модулей](../o-dokumentacii/slovar-terminov-3.md) системы;
- справа — набор настраиваемых [виджетов](#tab1) («Пользователи», «Статистика», «Отчет по категориям» и т. д.). Если открыт модуль, в правой части страницы отображается окно данного модуля;
- в правом верхнем углу — [имя пользователя](#tab2), при нажатии на которое можно перейти в личный кабинет и сторонние сервисы.

![Главная страница веб-интерфейса ИКС](../../images/article_22/homepage1-12.2.png)

## Настройка виджетов

Некоторые [виджеты](../o-dokumentacii/slovar-terminov-3.md) первоначально скрыты, но можно настраивать их, как удобно: скрывать и добавлять на главную страницу, изменять размер и перемещать по сетке.

Для перехода к настройкам нажмите ![Редактирование](../../images/article_22/redact.png) в правом нижнем углу — включится режим редактирования главной страницы.

![Режим редактирования главной страницы](../../images/article_22/homepage2-12.2.png)

Снизу откроется панель со всеми доступными виджетами. Отображаемые виджеты будут подсвечены.

Для **добавления виджета** просто нажмите на его иконку — он отобразится в самом низу страницы. Чтобы **переместить виджет**, зажмите его левой кнопкой мыши и перетащите.

**Изменить размер виджета** можно при помощи кнопки ![Настройки](../../images/article_22/settings.png) — в открывшемся окне введите нужные высоту и ширину. Задаваемые значения указываются в количестве клеток. Максимальная ширина одного виджета 36 клеток, что соответствует размеру отображаемой сетки, а максимальная высота — 3600 клеток. Изменить размер виджета возможно только при наличии достаточного места. Размер будет рассчитываться от левого верхнего угла и текущего расположения виджета. Если необходимо увеличить виджет, а свободного места не хватает, перетащите его в самый низ и введите необходимые размеры.

> ⚠ В мобильном виде виджеты располагаются по вертикали друг за другом, изменение расположения и размеров невозможно.

Для **удаления виджета** воспользуйтесь одним из способов:

- нажмите на нем ![Удаление](../../images/article_22/delete.png);
- зажмите его левой кнопкой мыши и перетащите на нижнюю панель.

Кнопкой ![Восстановить](../../images/article_22/reestablish.png) на нижней панели можно восстановить первоначальную настройку главной страницы.

Чтобы выйти из режима редактирования, нажмите ![Закрыть](../../images/article_22/close.png).

**Перечень виджетов**

| Виджет | Какую информацию показывает | Возможные действия | Куда можно перейти | Зависимые службы |
|--------|----------------------------|-------------------|------------------|-----------------|
| Пользователи | Общее количество, количество активных, заблокированных и отключенных пользователей | Добавить пользователя, импортировать пользователей | Пользователи, Роли, Монитор соединений | - |
| Статистика | Размер входящего и исходящего трафика за текущие день, неделю и месяц | Включить/отключить службу статистики (Статистика, Счетчики) | Отчеты | Статистика, Счетчики, [Прокси-сервер](../o-dokumentacii/slovar-terminov-3.md) |
| Отчеты по категориям | Сводная информация о распознанных категориях трафика за текущий день | - | Отчеты (Категории трафика) | - |
| Безопасность и ограничение доступа | Число обнаруженных атак и вирусов | Включить/выключить систему безопасности ([Межсетевой экран](../o-dokumentacii/slovar-terminov-3.md), Прокси-сервер, фильтр [HTTP](../o-dokumentacii/slovar-terminov-3.md)-трафика) | Наборы правил, Категории трафика, Межсетевой экран (правила), Прокси (настройки) | Межсетевой экран, Прокси-сервер, фильтр HTTP-трафика, [DNS-сервер](../o-dokumentacii/slovar-terminov-3.md), Детектор атак Suricata, Антивирусный прокси-сервер, Контент-фильтр |
| Провайдеры и сети | Провайдер по умолчанию, [ping](../o-dokumentacii/slovar-terminov-3.md) до шлюза по умолчанию, средняя загрузка интерфейса провайдера, количество [VPN](../o-dokumentacii/slovar-terminov-3.md)-подключений | Запустить мастер настройки сети | Провайдеры и сети, VPN (Пользователи), Управление отчетами (отчет по интерфейсам), Сетевые утилиты | - |
| Почта | Количество обработанных писем (отправленных, полученных, отправленных в [спам](../o-dokumentacii/slovar-terminov-3.md), обнаруженных вирусов) | Включить/выключить почтовые службы (Почтовый сервер, Хранилище почты, Сборщик почты, Антиспам SpamAssassin), Добавить почтовый ящик | Почта: главная страница, домены и ящики, фильтры, адресная книга, почтовая очередь, статистика почты | Почтовый сервер, Хранилище почты, Сборщик почты, Антиспам SpamAssasin, DNS-сервер, Сервер каталогов, Служба [DKIM-подписи](../o-dokumentacii/slovar-terminov-3.md) |
| Веб и файловый сервер | Количество виртуальных хостов, ресурсов, оставшееся свободное место на диске | Включить/выключить файловый сервер ([Веб-сервер](../o-dokumentacii/slovar-terminov-3.md), [FTP-сервер](../o-dokumentacii/slovar-terminov-3.md), Сетевое окружение), добавить один из ресурсов, добавить раздел и подключить жесткий диск | Хранилище файлов, Веб (Веб-ресурсы), [FTP](../o-dokumentacii/slovar-terminov-3.md) (FTP-ресурсы), Сетевое окружение (идентификация) | Веб-сервер, FTP-сервер, Сетевое окружение, DNS-сервер |
| Телефония | Количество внешних каналов, телефонных номеров, факсов | Включить/выключить службу телефонии, Добавить телефонный номер, добавить факс | Телефония (телефонные номера, правила, журнал звонков) | Телефония, DNS-сервер |
| Мониторинг и обслуживание | Загрузка процессора, оперативной памяти, системы. Время до конца демо-версии (если система не активирована) | Включить/выключить мониторинг системы, создать резервную копию | Мониторинг, Обновления, Время и дата, Системный журнал | Мониторинг состояния системы, Системные уведомления |
| Лента сайтов | Последние зарегистрированные посещения сайтов | - | Отчеты (Активность пользователей) | - |
| Лента поисковиков | Последние зарегистрированные запросы в поисковых системах | - | Отчеты (Лента поисковиков) | - |
| События | Последние события системы из системного журнала | - | Журнал и уведомления (Системный журнал) | - |

Все виджеты отображают актуальную информацию и обновляются в реальном времени.

> ⚠ Внимание! Без полной расшифровки трафика с подменой сертификата лента поисковиков не будет отображаться.

## Личный кабинет

Чтобы перейти в личный кабинет и сторонние сервисы, нажмите на имя пользователя в верхнем правом углу страницы и выберите нужный пункт в списке:

- Личный кабинет;
- [Xauth](../polzovateli-i-statistika/server-avtorizacii-xauth/server-avtorizacii-xauth-obzor-2.md);
- [Captive Portal](../polzovateli-i-statistika/captive-portal/captive-portal-obzor-2.md);
- [Веб-почта](../pochta/vebpochta/vebpochta-obzor-2.md);
- [Xphone](https://doc.a-real.ru/index.php?article=101).

![Личный кабинет](../../images/article_22/homepage3-12.2.png)

Данная кнопка используется также для [выхода](vyhod-iz-vebinterfeysa-2.md) из-под учетной записи ИКС.
