# Установка с флешки

ИКС позволяет записать образ на флеш-накопитель и выполнить установку с него.

---

Кроме установки с диска, ИКС позволяет записать образ на флеш-накопитель и выполнить установку с него. Для этого выполните следующие действия:

1. Для получения дистрибутива программы заполните специальную форму на [сайте](https://xserver.a-real.ru/#sect-download). На почту придет письмо со ссылками для скачивания. Перейдите по нужной ссылке и скачайте дистрибутив (с расширением img).

2. Запишите образ любой утилитой для создания загрузочных носителей (например, Rufus).

3. Поместите флеш-накопитель в USB-порт устройства (компьютера, ноутбука, планшета), на которое будет установлен ИКС. В BIOS укажите USB-flash в качестве первичного загрузочного устройства.

   > ⚠ Внимание! Не все модели материнских плат поддерживают установку с USB-flash-устройств.

4. Когда образ загрузится с накопителя, начнется установка в [обычном режиме](/index.php?article=269#step2).
