# Установка ИКС на Hyper-V

Microsoft Hyper-V — это система аппаратной виртуализации для x64-систем на основе гипервизора. Ранее была известна как виртуализация Windows Server (Windows Server Virtualization).

---

Для того чтобы установить ИКС на виртуальную платформу, выполните следующие действия:

1. Запустите гипервизор.

2. Выберите **Создать &gt; Виртуальная машина**.

3. Введите имя виртуальной машины и нажмите **«Далее»**.

4. Укажите количество оперативной памяти для работы.

5. Выберите виртуальную сеть, к которой будет подключена машина, и определите размер жёсткого диска.

6. Мастер запросит источник установки системы. Выберите **«Установить с загрузочного диска»**, **«Файл ISO-образа»** и укажите файл-образ ИКС, скачанный с нашего [сайта](https://xserver.a-real.ru/#download).

7. Нажмите **«Далее»**.

8. Кликните **«Готово»** — виртуальная машина будет создана.

9. После создания машины необходимо добавить ей ещё один сетевой адаптер. У выделенной машины нажмите **«Параметры»**.

10. В окне редактирования выберите **Установка оборудования &gt; Сетевой адаптер** и присвойте его той же виртуальной сети.

11. Нажмите **«ОК»**.

12. Кликните правой кнопкой мыши по нужной машине, а затем — **«Подключить»**. Машина будет запущена в отдельном окне.

13. Нажмите **Действие &gt; Пуск**. Машина запустится и начнётся установка ИКС.
