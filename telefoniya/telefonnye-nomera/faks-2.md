# Факс

Объект **«Факс»** служит для приема факсов на указанный номер и сохранения их в формате `.tif` в папке [хранилища файлов](https://doc.a-real.ru/index.php?article=80).

Чтобы добавить факс, выполните следующие действия:

1. Перейдите в меню **Телефония > Телефонные номера**.
2. Нажмите на папку с телефонными номерами, а затем — на кнопку **«Добавить»** и выберите **«Факс»**.

![Иллюстрация](../../../images/article_243/tel_num2.png)

3. Введите **номер** факса.
4. Если требуется, добавьте **описание**, которое будет отображаться в [правилах телефонии](https://doc.a-real.ru/index.php?article=104) и [журнале звонков](https://doc.a-real.ru/index.php?article=106) рядом с соответствующим номером.

![Иллюстрация](../../../images/article_243/fax1.png)

5. Укажите папку для сохранения TIF-файлов.
6. При установке флага **«Отправлять на почту»** задайте адрес, на который будут дублироваться TIF-файлы.
7. Нажмите **«Добавить»** — новый объект появится в списке.

## Отправить факс

Для того чтобы послать файл по факсу на внешний номер, выполните следующие действия:

1. На странице модуля **«Телефонные номера»** нажмите кнопку **«Отправить факс»**.
2. В открывшемся окне выберите **документ** для отправки. По факсу можно отправлять файлы следующих форматов: bmp, jpg, jpeg, gif, pdf, pdf многостраничный, tif, tif многостраничный, tiff, tiff многостраничный (вне зависимости от регистра).

![Иллюстрация](../../../images/article_243/fax2.png)

3. Укажите **внешний канал**.
4. Введите **номер**, на который необходимо отправить файл.
5. Нажмите **«Ок»** — файл будет отправлен на указанный номер.

> ⚠ Внимание! При отправке сразу нескольких факсов с одного сервера на другой некоторые факсы могут прийти частично. Чтобы решить проблему, достаточно повторно отправить факс, который пришел частично. При отсутствии негативных факторов факс придет целиком.
