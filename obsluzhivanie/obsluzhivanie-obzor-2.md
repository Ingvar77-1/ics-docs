В Шлюзе безопасности ИКС предусмотрены различные инструменты для обслуживания системы:

**Система**

Позволяет указать основные сведения о [системе](/index.php?article=107) (название и имейл-адрес организации, доменное имя), а также настроить удаление данных и просмотреть текущие задачи в ИКС.

**Все службы**

Отображает список всех [служб](/index.php?article=108) ИКС с возможностью управления ими.

**Резервные копии**

Предназначен для создания и удаления [копий](/index.php?article=109) данных в ИКС, а также для управления этими копиями.

**Управление питанием**

Обеспечивает [настройку](/index.php?article=110) источников питания и управление ими.

**Жесткие диски**

Содержит список всех [жестких дисков](/index.php?article=111), которые физически подключены к оборудованию с установленным ИКС. Предоставляет возможность управлять жесткими дисками.

**Время и дата**

Предназначен для установки параметров [времени и даты](vremya-i-data.md) в ИКС.

**Настройки интерфейса**

Предоставляет возможность установить параметры [веб-интерфейса](/index.php?article=113).

**Мониторинг**

Позволяет [просматривать](/index.php?article=114) статистику использования сетевых и системных ресурсов, различных показателей системы (виртуальная память, загрузка процессора и т. д.).

**Кластер**

[Обеспечивает](/index.php?article=423) отказоустойчивость системы и резервирование настроек.

**Журнал и уведомления**

Содержит список [сообщений](/index.php?article=115) о действиях пользователей, изменениях статусов сервисов, а также о системных ошибках. Позволяет фильтровать сообщения по различным параметрам (например, по категориям и дате).

**Техподдержка**

Предоставляет сотруднику службы [технической поддержки](/index.php?article=116) «А-Реал Консалтинг» доступ к ИКС для помощи при работе с программой.

**Обновления**

Отслеживает выходящие [обновления](/index.php?article=117) ИКС, отображает доступность и готовность к их установке.

**О программе**

Показывает основные сведения о [программе](/index.php?article=118), контактные данные службы технической поддержки, предоставляет возможность ознакомиться с лицензионным соглашением.
