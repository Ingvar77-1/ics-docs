# Время и дата

Модуль «Время и дата» предназначен для установки параметров времени и даты в ИКС. Для открытия модуля перейдите в меню **Обслуживание > Время и дата**.

---

Модуль **«Время и дата»** предназначен для установки параметров времени и даты в ИКС. Для открытия модуля перейдите в меню **Обслуживание > Время и дата**.

![Иллюстрация](../../images/article_112/time_date.png)

В модуле расположены следующие вкладки:

- [Время и дата](#tab1)
- [Сервер времени](#tab2)
- [Временные промежутки](#tab3)
- [Журнал](#tab4)

## Время и дата

На данной вкладке обязательными полями являются **«Дата»** и **«Время»**. Если сервер времени выключен, то можно задать необходимые значения времени и даты вручную.

![Иллюстрация](../../images/article_112/time_date1.png)

Поле **«Временная зона»** предназначено для указания текущей временной зоны.

Поле **«Сервера для синхронизации NTP-сервера»** содержит предустановленные URL пулов NTP-серверов. В данном поле можно указывать IP-адреса NTP-серверов или иных пулов. Встроенный NTP-сервер в ИКС использует данные URL для синхронизации времени и даты.

При установке флага в поле **«Автоматически создавать разрешающее правило для доступа к серверу времени»** будет выполнена установка разрешающего правила межсетевого экрана на доступ от локальных и DMZ-сетей на порт NTP (123).

После изменения временной зоны ИКС выдаст сообщение: «Временная зона была изменена. Чтобы изменения вступили в силу для всех служб, необходимо перезагрузить ИКС». Пользователь с ролью Администратор может нажать отмену, тогда система будет работать по новой временной зоне, но некоторые службы продолжат работу по старой.

![Иллюстрация](../../images/article_112/time_date1-2.png)

При новом открытии веб-интерфейса вновь покажется сообщение о необходимости перезагрузить ИКС.

## Сервер времени

На данной вкладке отображаются следующие сведения:

- статус сервера времени (**запущен**, **остановлен**, **выключен**, **не настроен**);
- текущее состояние сервера времени (NTP-сервера ИКС). Это URL NTP-сервера, с которым синхронизирован ИКС (0.0.0.0 означает, что сервера NTP не доступны), а также уровень точности данной синхронизации (1 — максимальная точность, 16 — сервера не доступны);
- кнопка **«Включить»** (**«Выключить»**) — позволяет запустить или остановить сервер;
- журнал последних событий.

![Иллюстрация](../../images/article_112/time_date2.png)

## Временные промежутки

Данная вкладка позволяет задать временные промежутки по дням недели и времени **«с… по…»**.

![Иллюстрация](../../images/article_112/time_date3.png)

На вкладке отображаются все созданные временные промежутки. Их можно добавлять, удалять и редактировать при помощи соответствующих кнопок. Созданные временные промежутки можно применять в различных модулях ИКС, где требуется задавать время действия.

Для создания нового временного промежутка выполните следующие действия:

1. Нажмите кнопку **«Добавить»**.
2. В открывшемся окне укажите название промежутка.
3. Установите дни недели при помощи флагов.
4. Выберите время действия.

![Иллюстрация](../../images/article_112/time_date3-2.png)

5. Нажмите кнопку **«Добавить»** — новый временной промежуток появится в списке.

## Журнал

На данной вкладке отображается сводка всех системных сообщений модуля с указанием даты и времени.

![Иллюстрация](../../images/article_112/time_date4.png)

Журнал является стандартным элементом веб-интерфейса ИКС.
