Файловый сервер Шлюза безопасности ИКС предназначен для удаленного хранения и передачи большого количества информации.

---

Файловый сервер Шлюза безопасности ИКС предназначен для удаленного хранения и передачи большого количества информации. Эти функции обеспечиваются следующими инструментами:

**Хранилище файлов**

Содержит список всех пользовательских [ресурсов](hranilische-faylov/hranilische-faylov-obzor.md), расположенных на ИКС. В модуле можно создавать следующие типы ресурсов: веб-ресурс, виртуальный хост, FTP-ресурс, общий ресурс.

**Веб**

Модуль предназначен для [настройки и управления](veb/veb-obzor-2.md) веб-сервером. В модуле можно создавать такие типы веб-ресурсов, как: веб-ресурс, виртуальный хост, виртуальный хост с перенаправлением, ссылка на виртуальный хост, а также настраивать работу по сертификату LetsEncrypt и управлять базами данных добавленных ресурсов.

**FTP**

Предназначен для [размещения файлов](ftp.md) на сервере и предоставления доступа к ним по сети.

**Сетевое окружение**

Отвечает за [обмен данными](setevoe-okruzhenie-2.md) в локальной сети. Модуль предназначен для настройки и управления удаленным доступом.
