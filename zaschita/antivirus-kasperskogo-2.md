# Антивирус Касперского

Модуль «Антивирус Касперского» предназначен для защиты от проникновения вредоносных программ в локальную сеть. Для открытия модуля перейдите в меню **Защита &gt; Антивирус Касперского**.

![Иллюстрация](../../images/article_68/kasper_av1-12.2.png)

В модуле расположены следующие вкладки:

- Антивирус Касперского
- Настройки
- События
- Журнал

## Антивирус Касперского

На данной вкладке отображается состояние службы «Антивирус Касперского»:

- статус службы (запущен, остановлен, выключен, не настроен);
- кнопка **«Включить»** (**«Выключить»**) — позволяет запустить или остановить службу;
- текущие версии базы и антивируса Касперского;
- дата истечения лицензии;
- журнал последних событий.

![Иллюстрация](../../images/article_68/antivir_kaspersky1.png)

Служба **«Антивирус Касперского»** отвечает за работоспособность предустановленного Антивируса Касперского, который проверяет почтовые письма (если установлен соответствующий флаг в настройках [почтового сервера](/index.php?article=85#antivirus)), а также трафик HTTP и HTTPS (если установлен флаг в настройках [прокси-сервера](/index.php?article=62)).

> ⚠ Внимание! По умолчанию служба находится в состоянии «не настроен». Чтобы активировать ее, установите в настройках [прокси-сервера](/index.php?article=62) флаг **«Использовать антивирус Касперского»**. При этом у вас должен быть приобретен лицензионный ключ.

## Настройки

Данная вкладка предназначена для настройки работы Антивируса Касперского.

Кнопка **«Менеджер лицензий»** позволяет загружать и просматривать сведения о файле лицензии антивируса.

![Иллюстрация](../../images/article_68/kasper_av2-12.2.png)

Флаги **«Использовать в прокси»** и **«Проверять почту»** соответствуют аналогичным флагам в настройках [прокси](/index.php?article=62) и [почтового сервера](/index.php?article=85). Данные флаги включают Антивирус Касперского для фильтрации трафика, проходящего через прокси-сервер и вложения в почтовых письмах. При включении данных флагов можно также установить флаг **«Расширенное журналирование событий»** — тогда в [журнале](#tab4) будет отображаться дополнительная информация о работе модуля.

> ⚠ Внимание! Опция расширенного журналирования может значительно нагружать систему.

Флаг **«Обновлять базы при выключенном антивирусе»** позволяет запускать проверку обновления баз при выключенной службе «Антивирус Касперского». При значительной нагрузке на систему антивирус может обновляться только во время отключения.

Поле **«Проверять наличие обновлении баз антивируса»** позволяет задать частоту проверок обновлений баз антивируса. По умолчанию этот процесс будет запускаться каждый час (если на данной вкладке установлен хотя бы один флаг). При выборе периода «Каждый день» или «Раз в неделю» можно указать время начала проверки.

![Иллюстрация](../../images/article_68/kasper_av3-12.2.png)

Выполните **настройки сканирования**:

- Эвристический анализ (не использовать, минимальный, средний, детализированный и максимальный уровни);
- Включение/отключение сканирования архивов;
- Включение/отключение сканирования исполняемых файлов;
- Максимальное время ожидания сканирования;
- Включение/отключение лечения, если обнаружен вирус;
- Включение/отключение статуса заражен, если объект предположительно заражен;
- Включение/отключение статуса заражен, если сканирование невозможно.

Введите **сообщение о запрете доступа в прокси**, которое будет выдаваться при обнаружении вируса.

Также предусмотрена возможность отредактировать **HTML-страницу запрета доступа**. Нажмите на соответствующую кнопку, измените код в открывшемся окне и сохраните изменения.

![Иллюстрация](../../images/article_68/kasper_av6-12.2.png)

Чтобы изменения настроек антивируса вступили в силу, нажмите **«Сохранить»**.

> ⚠ Внимание! Для корректной работы антивируса необходимо настроить [расшифровку HTTPS-трафика](/index.php?article=168).
