Шлюз безопасности ИКС предоставляет множество инструментов для управления пользователями и их правами, а также для настройки авторизации пользователей и контроля статистики.

---

**Пользователи**

Позволяет [создавать](/index.php?article=124) пользователей (группы пользователей) различными способами, а также управлять ими и назначать права доступа.

**Роли**

Предназначен для [установки](/index.php?article=44) прав доступа пользователям ИКС к программе.

**Наборы правил**

Позволяет [создавать](/index.php?article=45) наборы правил для объединения различных правил, прикрепленных к определенному пользователю, группе либо роли, а также для объединения правил определенной сферы деятельности (например, набор правил для школ).

**Категории трафика**

Предназначен для [объединения](/index.php?article=46) множества IP-адресов и URL в единую категорию, которая может применяться при создании правил прокси-сервера.

**Монитор соединений**

Позволяет [контролировать](/index.php?article=47) потоки трафика и просматривать все соединения ИКС и пользователей.

**Диапазоны адресов**

Предоставляет [доступ](/index.php?article=48) к списку всех объектов тарификации, по которым ИКС ведет учет. Позволяет добавлять пользовательские диапазоны адресов.

**Отчеты**

Позволяет формировать и просматривать различные [отчеты](/index.php?article=49), такие как сводный отчет, категории трафика, активность пользователей и другие.

**Управление отчетами**

Предоставляет возможность [управлять](/index.php?article=144) и настраивать службы «Статистика» и «Счетчики», а также формировать отчеты о работе данных служб.

**Сервер авторизации (Xauth)**

Обеспечивает [настройку](/index.php?article=51) и работу службы авторизации пользователей Xauth.

**Настройки авторизации**

Позволяет [настроить](/index.php?article=190) соединение с доменом в сети при помощи сетевого протокола Kerberos либо LDAP.

**Captive Portal**

Обеспечивает [настройку](/index.php?article=52) и работу авторизации пользователей на ИКС для доступа к сети Интернет при помощи следующих типов авторизации: веб, по SMS, по звонку.

**Синхронизация**

Предназначен для настройки [синхронизации](/index.php?article=53) пользователей ИКС с AD.
