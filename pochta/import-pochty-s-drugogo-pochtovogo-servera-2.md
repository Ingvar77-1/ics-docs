# Импорт почты с другого почтового сервера

[Почтовый сервер](https://doc.a-real.ru/index.php?article=84) ИКС использует для хранения содержимого почтовых ящиков формат [Maildir](https://doc.a-real.ru/index.php?article=24#maildir).

Чтобы перенести содержимое своего почтового сервера на ИКС, выполните следующие действия:

1. Сохраните каждый почтовый ящик в формате Maildir.

2. Скопируйте ящики на ИКС при помощи [FTP](https://doc.a-real.ru/index.php?article=24#ftp) или [SMB](https://doc.a-real.ru/index.php?article=24#smb)-протокола в хранилище файлов.

3. В меню **Почта > Домены и ящики** добавьте те же самые [почтовые ящики](https://doc.a-real.ru/index.php?article=87#add3), которые сохранили в **Шаге 1**.

4. Свяжитесь со [службой технической поддержки ИКС](https://xserver.a-real.ru/#main-footer) — технические специалисты перенесут содержимое ящиков в директорию почтового сервера.
