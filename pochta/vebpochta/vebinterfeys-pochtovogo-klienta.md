Для того чтобы пользователи могли получать и отправлять письма с почтового сервера ИКС без применения сторонних клиентов, можно воспользоваться встроенным почтовым веб-интерфейсом Roundcube. Изменение параметров работы и внешнего вида Roundcube осуществляется в настройках модуля «Веб-почта».

---

Перейти в веб-интерфейс почтового клиента Roundcube можно одним из способов:

- перейдите по ссылке «Веб-почта» из окна авторизации (если в настройках интерфейса не установлен флаг «Скрывать ссылку на веб-почту в окне авторизации»);

![Иллюстрация](../../../images/article_236/start-9.0.jpg)

- наберите в адресной строке браузера конструкцию вида `адрес_ИКС:порт_веб_почты/rc`.

![Иллюстрация](../../../images/article_236/roundcube2.png)

<span style="background-color: #fbeeb8;">Веб-интерфейс Roundcube доступен только по протоколу [HTTPS](/index.php?article=24#https).</span>

Roundcube позволяет полноценно оперировать сообщениями пользователя, создавать папки, фильтры, настраивать адресную книгу и т. д. В целом веб-интерфейс похож на почтовый интерфейс популярных почтовых сервисов, таких как yandex или gmail.

В веб-интерфейсе расположены следующие разделы:

- [Почта](#tab1)
- [Контакты](#tab2)
- [Настройки](#tab3)

## Почта

В данном разделе хранятся и отправляются письма: входящие и отправленные письма, черновики, спам, письма в корзине.

![Интерфейс раздела Почта](../../../images/article_236/roundcube3.png)

При создании письма во время набора адреса получателя интерфейс ищет совпадающие адреса в адресной книге и предлагает их на выбор.

В интерфейсе предусмотрена возможность отметить письмо как **спам**. Для этого необходимо нажать на него правой кнопкой мыши и выбрать «Переместить в... > СПАМ».

![Меню перемещения в спам](../../../images/article_236/spam.png)

## Контакты

В данном разделе собраны имейл-адреса контактов пользователя.

![Раздел Контакты](../../../images/article_236/roundcube4.png)

В разделе расположены следующие **вкладки**:

- «Персональные адреса», куда можно сохранять любое количество своих деловых контактов;
- «Собранные адреса» — это адреса, которые Roundcube добавил в процессе работы (например, когда вы указываете в получателе новый адрес, Roundcube запоминает его);
- «Адресная книга» — содержит список тех пользователей и их почтовых ящиков, которые заведены на ИКС. При выборе пользователя в книге отображаются имейл-адреса, которые ему назначены. При необходимости можно импортировать контакты адресной книги, а также заводить новые. Список адресов можно импортировать в файл формата VCard нажатием одноименной кнопки.

## Настройки

Данный раздел предназначен для установки параметров работы почтового клиента.

![Раздел Настройки](../../../images/article_236/roundcube5.png)

В разделе расположены следующие **вкладки**:

- «Настройки» — содержит необходимые опции для удобства работы с системой. В основном они аналогичны настройкам любого почтового клиента: можно указать частоту проверки новых сообщений, кодировку писем по умолчанию, параметры хранения и удаления писем и т. д.
- «Папки» — позволяет создавать дополнительные ярлыки для хранения писем и переноса их туда при помощи фильтров.
- «Профили» — содержит данные пользователя, в том числе его подпись.
- «Ответы» — позволяет создавать автоматические ответы на письма.
- «Ключи PGP» — предназначена для создания пар ключей шифрования [PGP](/index.php?article=24#pgp), которые потом можно использовать при отправке писем.
- «Фильтры» — позволяет [управлять](/index.php?article=359) пересылкой в автоматическом режиме.

> ⚠ Внимание! Почтовый ящик можно удалить и создать заново с использованием сохраненного бекапа. Фильтры такого пересозданного почтового ящика останутся, но будут выключены.

Более подробно о работе веб-интерфейса Roundcube можно прочитать [здесь](https://roundcube.net/).

В ИКС при помощи почтового клиента также можно [настроить миграцию почты со стороннего домена](/index.php?article=412).
