![Иллюстрация](../../../images/article_94/mail_stat-1.png)

Модуль **«Статистика и очередь»** предназначен для формирования статистических данных почтового сервера ИКС, а также для управления почтовой очередью. Чтобы открыть модуль, перейдите в меню **«Почта > Статистика и очередь»**.

В модуле расположены следующие вкладки:

- [Статистика](#статистика)
- [Почтовая очередь](#почтовая-очередь)

## Статистика

Данная вкладка предназначена для вывода статистики почтового сервера ИКС.

В левой части вкладки расположены инструменты для выбора параметров, по которым будет выводиться статистика. В правой части вкладки выводятся результаты статистики в зависимости от заданных параметров.

![Иллюстрация](../../../images/article_94/mail_stat1.png)

Флаг **«Детализация писем»** предназначен для просмотра всех писем за период. При установке данного флага поле «Группировка» становится неактивным.

Поле **«Группировка»** предназначено для формирования отчета по времени (по месяцам, дням, часам), доменам (по доменам отправителя, по доменам получателя) либо ящикам (по ящикам отправителя, по ящикам получателя). Возможен выбор только одного значения для группировки записей.

В поле **«Отправитель»** можно задать часть или полное значение почтового адреса отправителя.

Поле **«Получатель»** позволяет задать часть или полное значение почтового адреса получателя.

Поле **«IP-адрес»** позволяет задать фильтрацию по IP-адресу. Доступен ввод нескольких адресов.

Поле **«Статус»** представляет собой выпадающий список с возможными статусами писем. Доступен выбор нескольких статусов.

В полях **«Время с»** и **«по»** можно указать временной промежуток для фильтрации записей.

После установки всех необходимых параметров нажмите кнопку **«Показать»** — результаты статистики будут представлены справа в виде таблицы. Столбцы таблицы варьируются в зависимости от применяемого фильтра.

Выбор **временного периода** отображения данных осуществляется так же, как в **стандартном журнале** ИКС.

При необходимости можно сохранить данные статистики почты в файл. Для этого нажмите кнопку **«Экспорт»** и выберите нужный формат файла (`.csv`, `.txt` или `.xls`).

## Почтовая очередь

На данной вкладке показаны почтовые сообщения, ожидающие отправки. Это могут быть как вновь попавшие в почтовую очередь сообщения, так и повторно (например, сообщения, отклонённые **серым списком** вышестоящего почтового сервера).

При выборе любого объекта из списка можно увидеть код ошибки (если почтовое сообщение попало в очередь повторно), по которой письмо не было доставлено. Также при помощи соответствующих кнопок можно **скачать** письмо в формате `*.eml`, попытаться **отправить** его или **удалить** из очереди.

Управлять почтовой очередью можно при помощи кнопок **«Очистить очередь»** и **«Отправить все»**.

![Иллюстрация](../../../images/article_94/mail_stat2.png)

Установить **параметры работы** почтовой очереди можно в меню **«Почта > Настройки»**.
